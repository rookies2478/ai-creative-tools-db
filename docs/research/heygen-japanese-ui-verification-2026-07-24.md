# HeyGen 日本語UI検証記録（2026-07-24）

## 対象
監査報告書 `docs/audits/db-content-consistency-audit-2026-07-24.md` AUD-07（P1）
- 事象: `src/pages/categories/avatar-video/index.astro`(58) で HeyGen の `ja` が `'対応'` と断定表示されていた。同じ表内の他3ツール（Synthesia/D-ID/InVideo AI）は全て `'要確認'` であり、表記が不統一だった。
- DB (`src/content/tools/heygen.md`) の `japaneseUi` は元々 `"unknown"` で、本文・FAQ・limitations等でも「日本語UIの対応状況は公式情報で明記されていない（要確認）」と一貫して記載されていた。
- 結論: DB側は正しい。ページ側（カテゴリ一覧のハードコード値）のみが誤りだった。

## 確認した公式情報（2026-07-24時点）
一次情報中心で確認。断定材料が不足する部分は二次情報で補い、その旨明記する。

1. HeyGen公式ヘルプセンター（help.heygen.com）トップページ
   - カテゴリ構成: Avatars / Voice / Translation / Video / Account + Billing / Apps / About HeyGen
   - 「Interface language」「UI language」を明示的に扱う記事は確認できなかった（トップナビ範囲内での確認、個別記事は未閲覧）。
2. HeyGen公式 Dashboard Overview / Platform Overview ページ（heygen.com/academy/platform-overview）
   - ダッシュボードのナビゲーション・機能説明ページだが、UI表示言語の切替設定については言及がない。
   - ページ自体が日本語で表示され「日本語」の言語セレクタが見えるのは、公式サイト（マーケティング/ヘルプページ）のローカライズであり、製品ダッシュボードのUI言語設定の証拠にはならない（別物）。
3. 二次情報（未一次確認・参考程度）
   - Chrome ウェブストアに「HeyGen: interface translator」という非公式拡張機能があり、Google翻訳APIを使ってHeyGenのUI全体を133言語に機械翻訳する、と説明されている。これはブラウザ側の自動翻訳であり、HeyGen公式が提供するUI言語設定ではない。
   - この拡張機能の存在自体が、逆説的に「公式のネイティブ日本語UI切り替えは無い（少なくとも一般的ではない）」ことを示唆する状況証拠にはなるが、確定情報ではない。
